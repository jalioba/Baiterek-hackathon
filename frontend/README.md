# 🌳 Цифровой портал «Байтерек» — Единое окно господдержки бизнеса
### Разработано для хакатона по заданию прототипа цифровой платформы «Единый портал поддержки бизнеса» (ЕППБ) на базе универсального конструктора
### Заказчик АО «НИХ «Байтерек»
## 🚀 О проекте
«Байтерек» — это инновационный цифровой портал, объединяющий все меры государственной поддержки малого и среднего бизнеса (МСБ) в Казахстане. Проект решает проблему фрагментированности государственных услуг, собирая кредиты, субсидии, гарантии, гранты и лизинг от всех дочерних организаций холдинга в одном современном и понятном интерфейсе.

## 💡 Ключевые Инновации (Идеи для презентации/Питча)
1. **Единое цифровое окно (One-Stop-Shop)**: Предпринимателю больше не нужно искать информацию на десятках разных сайтов. Все дочерние организации (БРК, QazIndustry, КазАгроФинанс, KazakhExport и др.) интегрированы в одну экосистему с единым личным кабинетом.
2. **Умный AI-Ассистент «Жакын AI»**: Встроенный искусственный интеллект, который анализирует профиль компании и предлагает наиболее релевантные меры поддержки в формате естественного диалога.
3. **Бесшовная локализация (i18n)**: Мгновенное переключение между русским (РУС) и казахским (ҚАЗ) языками без перезагрузки страниц, охватывающее все элементы интерфейса и динамические данные.
4. **Биометрическая верификация**: Инновационный и безопасный процесс регистрации с визуальной биометрической проверкой лица (проверка живости: "поверните голову вправо/влево") для предотвращения фрода и повышения доверия к платформе.
5. **Премиальный UX/UI Дизайн**: Использование современных паттернов (glassmorphism, градиенты, плавная анимация через Framer Motion), адаптированных под людей с ограниченными возможностями (версия для слабовидящих).

## 🛠 Технологический Стек

### Frontend (Клиентская часть)
- **Фреймворк:** Next.js 14 (App Router) — серверный рендеринг и оптимизация SEO.
- **Язык:** TypeScript — строгая типизация и минимизация багов.
- **Стилизация:** Tailwind CSS — utility-first подход для быстрого построения UI.
- **Анимации:** Framer Motion — плавные переходы между страницами и микроинтеракции.
- **Иконки:** Lucide React.
- **Графики:** Recharts — визуализация аналитики.
- **Управление формами:** React Hook Form + Zod (схемы валидации).

### Backend & Инфраструктура
- **База данных и Аутентификация:** Firebase (Firestore, Auth) — real-time обновления и надежное хранение данных.
- **Сервер:** Node.js (интеграция сторонних API и бизнес-логики).

## 📂 Архитектура и Структура Проекта
Проект спроектирован с учетом масштабируемости (Feature-Sliced Design элементы):

```text
frontend/
├── app/                  # Next.js App Router (маршруты приложения)
│   ├── (admin)/          # Панель администратора (Аналитика, Управление пользователями)
│   ├── (dashboard)/      # Личный кабинет (Мои заявки, Уведомления, Калькуляторы)
│   ├── (public)/         # Публичный контент (Каталог, Новости, База знаний)
│   └── layout.tsx        # Корневой макет с глобальными провайдерами (i18n, Theme, Auth)
├── components/           # Переиспользуемые UI компоненты
│   ├── auth/             # Многошаговый Wizard регистрации с биометрией
│   ├── features/         # Сложные бизнес-фичи (Чат-виджет Жакын AI, SmartSelector)
│   ├── home/             # Уникальные секции главной страницы (Счетчики, Как это работает)
│   ├── layout/           # Header, Footer, навигационные панели
│   └── shared/           # Базовые UI-компоненты (Карточки услуг, Кнопки, Статистика)
├── lib/                  # Утилиты (конфигурация Firebase, i18n словари, mock-данные)
├── hooks/                # Кастомные React-хуки (useAuth, useLiveMetrics, useLang)
└── public/               # Статические ресурсы (шрифты, логотипы, иллюстрации)
```

## 🗺 Подробное описание страниц (Сценарий презентации)

### 1. Главная страница (Landing)
Встречает пользователя премиальным дизайном.
- **Live-Счетчики:** Анимированные цифры показывающие масштаб холдинга (выданные средства, количество клиентов).
- **Блок "Как это работает":** 4 простых шага, объясняющих процесс от поиска до получения финансирования.
- **Доступность:** В шапке сайта встроена функция для изменения размера шрифта и контрастности (инклюзивность).

### 2. Каталог услуг (`/services`)
Ядро портала для поиска программ.
- **Smart Selector (Умный подбор):** Если пользователь не знает что искать, он нажимает одну кнопку, и система через серию наводящих вопросов сама фильтрует каталог.
- **Продвинутые фильтры:** Поиск по дочерней организации, срокам рассмотрения, типу бизнеса (ИП/ТОО). 

### 3. Личный кабинет предпринимателя (`/dashboard`)
Закрытая зона после прохождения биометрической регистрации.
- **Дашборд:** Инфографика (круговые/линейные диаграммы) текущих заявок.
- **История заявок:** Таймлайн со статусами (Подана -> На рассмотрении -> Документы -> Одобрено).
- **Калькуляторы:** Встроенные инструменты для предварительного расчета ежемесячных платежей по лизингу и кредитам.
- **Уведомления:** Центр сообщений для связи с менеджерами.

### 4. Информационный блок (`/news` и `/knowledge`)
- **Пресс-центр:** Консолидированная лента новостей всех 8 дочерних организаций с возможностью фильтрации по источнику.
- **База знаний:** FAQ и обучающие статьи по ведению бизнеса, экспорту и налогам.

### 5. Панель Администратора (`/admin`)
Рабочее место сотрудников холдинга.
- **Аналитика:** Глубокая статистика по популярности услуг, конверсии одобрений, притоку пользователей (с использованием Recharts).
- **Управление:** Инструменты для редактирования статусов заявок и модерирования платформы.

### 6. Модуль Регистрации (Инновация)
- **Многошаговый визард:** Выбор типа субъекта -> Личные данные -> Пароль -> **Биометрия лица** -> Подтверждение. На шаге биометрии пользователь видит интерфейс сканирования, который просит повернуть голову вправо и влево для подтверждения личности.

## 💻 Инструкция по запуску (Для разработчиков)

1. **Клонирование репозитория:**
   ```bash
   git clone <repo-url>
   cd Baiterek
   ```

2. **Запуск Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   Приложение будет доступно по адресу: `http://localhost:3000`

3. **Запуск Backend:**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

4. **Переменные окружения:**
   Скопируйте `.env.example` в `.env.local` и вставьте актуальные ключи от Firebase.
